SmartRecord

A focused macOS recorder for screen, system audio, microphone narration, SmartFocus zoom, H.264 MP4 export, and local Whisper VTT subtitles.

Download DMG View source

Version 1.03. Requires macOS with ScreenCaptureKit support.

Complete capture

Record the display, microphone, system audio, or any selected audio combination.

SmartFocus

Mouse-click focus zoom highlights the region being explained, then returns smoothly.

Local subtitles

Use the local Whisper medium model to generate WebVTT subtitle files after recording.

English

SmartRecord is a macOS screen recording app for tutorials, demos, and product walkthroughs. It captures screen video, microphone narration, system audio, and mouse focus events, then exports an H.264 MP4. You can optionally generate VTT subtitles locally with the Whisper medium model.
